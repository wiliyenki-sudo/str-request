1. Pendahuluan & Alur Proses

Aplikasi STR & ADJ Request dijalankan di dalam platform Lark sebagai H5 mini-app. Digunakan oleh tim site untuk mengajukan permintaan Stock Transfer Request (STR) dan Adjustment (ADJ) secara digital, lengkap dengan workflow approval dan ekspor CSV ke SAP.

Alur Proses STR

Site Buat STR→ Approval Manager→ ICO Processing→ Download CSV→ Done / SAP

Alur Proses ADJ

Site Buat ADJ→ ICO Processing→ Download CSV→ Done / SAP

ℹ️ Catatan

ADJ tidak melalui tahap Approval — langsung diproses oleh ICO setelah disubmit.

Role Pengguna

RoleAksesLayar yang Digunakan
Requestor (Site)Buat STR & ADJForm STR, Form ADJ
Approver (Dept Head / Manager)Approve atau Reject STRSTR List, STR Approval Detail
ICOProses STR/ADJ, download CSVICO STR Detail, ICO ADJ Detail

Gbr 8.1 — ICO ADJ Detail

📋 Alur ICO ADJ

  1. Buka ADJ berstatus Waiting ICO
  2. Isi kolom Article Doc untuk setiap baris item
  3. Isi Nomor Reservasi (wajib)
  4. Tekan Download CSV untuk mendapatkan file SAP
  5. Upload CSV ke SAP
  6. Tekan Process Done untuk menyelesaikan
  7. Status ADJ berubah ke Done

⚠️ Perhatian

Nomor Reservasi wajib diisi sebelum menekan Process Done. Tombol tidak akan aktif jika field kosong.

8.2 Format CSV ADJ (SAP Upload)

CSV ADJ menggunakan 7 kolom sesuai template upload SAP Adjustment:

Article,Discrep,Sloc1,Site,Sloc2,Remark,Mvt 1001234,-5,1009,J724,1009,RCP001,939 1001890,3,1000,J724,1000,RCP002,940
KolomSumber / Logika
ArticleKode artikel
DiscrepNilai Disc (Fisik − Sistem)
Sloc1Jika Sistem < Fisik (disc +) → 1000; jika Sistem > Fisik (disc −) → 1009
SiteKode site ADJ
Sloc2Jika Sistem > Fisik (disc −) → 1009; jika Sistem < Fisik (disc +) → 1000
RemarkNilai Receipt/Email dari form
MvtDisc negatif → 939; disc positif → 940

💡 Logika Sloc & Mvt

  • Disc + (Fisik > Sistem): Sloc1=1000, Sloc2=1000, Mvt=940
  • Disc − (Fisik < Sistem): Sloc1=1009, Sloc2=1009, Mvt=939

Lampiran A: Format Nomor Pengajuan

Baik STR maupun ADJ menggunakan format nomor yang sama:

SEQ / SITE / DEPARTMENT / MM / YYYY
BagianKeteranganContoh
SEQUrutan bulanan, 3 digit, reset tiap bulan001, 002, 015
SITEKode site pengajuJ724
DEPARTMENTNama department lengkapRUANG TAMU & PENYIMPANAN
MMBulan submit (2 digit)06
YYYYTahun submit2026
Contoh lengkap:
003/J724/RUANG TAMU & PENYIMPANAN/06/2026
Counter SEQ untuk STR dan ADJ berjalan terpisah. STR ke-3 di bulan Juni bisa bernomor 003, sementara ADJ ke-1 di bulan yang sama bernomor 001.
